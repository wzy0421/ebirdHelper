# ebirdHelper
TamperMonkey script for enhancing experience of ebird website.

The feature of this script include:
- Append Chinese common name after English common name.
- Highlight unseen species. All species not in your life list will be highlighted. One-country endemic species are highlighted using a different color. Every time you enter your species list page, your list will be updated.

This scripts relies on following data:
- ebird taxonomy.
- IOC multilingual version.
- Endemic species list from Checklist of the One-country Endemic Birds of the World from Lynx Edicions.

The taxomomy of these data sources are not fully aligned so there might be missing or misaligned.
